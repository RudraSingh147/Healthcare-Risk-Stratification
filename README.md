  🏥 Healthcare Risk Stratification System

   📌 Project Overview

Healthcare Risk Stratification System is a Machine Learning-based healthcare analytics application developed using Python and Streamlit. The system predicts patient risk levels based on demographic, clinical, and laboratory parameters.

The application helps healthcare professionals identify high-risk patients, analyze laboratory reports, perform risk simulations, and generate patient reports.

   📊 Machine Learning Model

Model Used:
- Logistic Regression

Input Features:
- Age
- Treatment Cost
- Abnormal Lab Count

Output:
- Low Risk
- Moderate Risk
- High Risk
